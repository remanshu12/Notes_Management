# Notes Management System - Frontend

React + Vite frontend using simple CSS.

## Run
```bash
cd frontend
npm install
npm run dev
```

Backend API expected at `http://localhost:8080/api`.

## Pages
- Dashboard
- My Notes
- Shared Notes
- Groups
- Notifications
- Version History

## Folder Structure
- `src/components`: reusable UI components
- `src/pages`: page-level screens
- `src/services/api.js`: Axios API functions
